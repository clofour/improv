# improv

Improv is a Hack Club YSWS centered around shipping provisioning toolings to receive infrastructure-related prizes and discounts. This repository contains the website for Improv. Currently, only the landing page is implemented, with more to come soon.

I created this project as I notice an infrastructure-size gap in Hack Club YSWS and I'd love to see more attention being given to it. Infrastructure is everwhere, and yet most Hack Clubbers don't know how to make their own. Instead, they rely on services like Vercel or Railway, creating significant knowledge gaps of what actually happens under the hood.

## Quick Start

To run the website, use `npm run dev`. To build the website, use `npm run build`.
